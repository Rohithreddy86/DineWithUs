About this Web Application

This SPA is developed in Angular with perfectly designed reusable components.For database and backend, We used serverless Firebase technology from google to store, retrieve and updata data in realtime. The data shown in this SPA will be realtime. For Authentication for admin users, we used firebase authentication module.To retrieve likes, views and comments of a particular video we used Youtube Data API v3 and iframes to view your desired youtube video in our SPA without having to redirect to youtube.
